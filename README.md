# 💻 Developer Portfolio

A modern, fully responsive **portfolio website** showcasing my work as a **Full-Stack Web Developer** — combining creative design with clean and efficient code. With "eastereggs"...

## 🚀 About
This project demonstrates my ability to build complete web applications — from frontend user interfaces to backend logic and API integration.  
The site is built with a focus on **performance**, **accessibility**, and **modern design**.

## 🧩 Tech Stack
- **Frontend:** HTML, CSS (custom + variables), JavaScript  
- **Backend:** PHP
- **Tools:** Git, VS Code

## 🎨 Features
- Dark mode with custom color palette  
- Animated sections and hover effects  
- Responsive layout for all devices  
- Clean, accessible design  
- Optimized CSS variables for easy theming
